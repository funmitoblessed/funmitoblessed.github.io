Every color is a mix of Red, Green, and Red. The RGB(x, y, z) represent the values for Red, Green, and Blue respectively. RGB(255, 0, 0) represents all RED (No Green or Blue). RGB(0, 255, 0) represents all GREEN (No Red or Blue). RGB(0, 0, 255) represents all BLUE (No Red or Green).
Each color can take any value between 0 and 255. The higher the value for a particular color, the more it resembles that color. If for instance you have RGB(200, 50, 234), it will resemble a mix of Red and Blue (which gives Magenta), So the color will be something close to magenta The region where red and blue meet is magenta. Red and Green is yellow. Blue and Green is Cyan The three colors together make white.
