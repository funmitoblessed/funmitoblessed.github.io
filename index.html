Memory Game

Arcade Game

Udacity Front-End Web Development Nanodegree Arcade Game Project. The aim of the game is to see how many times you can cross the stone paths and reach the water while avoiding being hit by the enemies on the screen within the set time (1 minute). Each crossing of the river earns you 1 point. See how many points you can gather before your time runs out. Use the direction arrows (up, down, right, left) to control the player in the same direction repectively. If you get hit by the enemy, your score reduces by 1. At the end of the game time, your score is returned to you.


Pixel Art Maker

Feed Reader Test

Udacity Front-End Web Development Nanodegree Feedreader test Project, a web-based application that reads RSS feeds and incorporates Four Tests Suites and Seven specs using Jasmine.


Restaurant Reviews

Udacity Front-End Web Development Nanodegree Restaurant Reviews Project. A static web page is converted to a mobile-ready web application. Initial design lacks accessibility and is made to be responsive on different sized displays and accessible for screen reader use. Some assets are also cached for offline use.
